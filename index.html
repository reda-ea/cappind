cappind.py by reda-ea

Zenity for appindicators

Tools like zenity and yad can easily make simple scripts completely graphical, by providing many GUI elements easily accessible by command line. However, none of them supports ubuntu appindicators (yet). This little script does just that.

Dependencies

python and ubuntu indicator libs

Install

just put the script somewhere in you path, and use it with your scripts
